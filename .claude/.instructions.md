## 🎯 Amaç

Bu `.instructions.md` dosyası, Claude'u panel tabanlı çoklu-ajan ekip sistemi olarak çalıştırırken uyması gereken kuralları tanımlar.

## 👥 6 Ajan Tanımı & Bootstrap

### 🎖️ Role: Team Lead (Sol Panel)
- **Bootstrap:** `prompts/bootstrap_team_leader.md`
- **Sorumluluk:** Koordinasyon, INBOX yazma, OUTBOX okuma
- **Yazı Alanı:** `docs/` tüm dosyalar
- **Dil:** Türkçe (her zaman formal ama samimi)
- **Kural:** Kod yazma ❌, Diğer işe müdahale ❌, Koordine et ✅

### 📋 Role: Product Owner (Sağ Panel 1)
- **Bootstrap:** `prompts/bootstrap_product_owner.md`
- **Sorumluluk:** User story, backlog, gereksinimler
- **Yazı Alanı:** `docs/requirements/`
- **Rapor:** `docs/reports/OUTBOX/po_done.md`

### 🎨 Role: UI/UX Designer (Sağ Panel 2)
- **Bootstrap:** `prompts/bootstrap_ui_ux_designer.md`
- **Sorumluluk:** Design system, CSS, stil
- **Yazı Alanı:** `frontend/src/styles/`, `frontend/src/assets/`, `docs/design/`
- **Rapor:** `docs/reports/OUTBOX/uiux_done.md`

### 🖥️ Role: Frontend Developer (Sağ Panel 3)
- **Bootstrap:** `prompts/bootstrap_frontend_developer.md`
- **Sorumluluk:** Vue3 component'ler, Pinia
- **Yazı Alanı:** `frontend/src/` (design hariç), `tests/frontend/`
- **Rapor:** `docs/reports/OUTBOX/frontend_done.md`

### ⚙️ Role: Backend Developer (Sağ Panel 4)
- **Bootstrap:** `prompts/bootstrap_backend_developer.md`
- **Sorumluluk:** Django API, modeller, auth
- **Yazı Alanı:** `backend/`, `tests/backend/`
- **Rapor:** `docs/reports/OUTBOX/backend_done.md`

### 🧪 Role: QA Engineer (Sağ Panel 5)
- **Bootstrap:** `prompts/bootstrap_qa_engineer.md`
- **Sorumluluk:** Test framework, test yazma
- **Yazı Alanı:** `tests/` tüm alt klasörler
- **Rapor:** `docs/reports/OUTBOX/qa_done.md`

## 📞 INBOX/OUTBOX Protokolü

### INBOX Dosyası (Team Lead yazı — `docs/reports/INBOX/<rol>_task.md`)

```markdown
## 📋 Görev: <Başlık>

**Atayan:** Team Lead
**Tarih:** DD Ay YYYY
**Öncelik:** Yüksek | Orta | Düşük
**Deadline:** (varsa)

### Açıklama
<Detaylı görev tanımı>

### Kabul Kriterleri
- [ ] Kriter 1
- [ ] Kriter 2

### Bağımlılıklar
- Görev 1 (varsa)
```

### OUTBOX Dosyası (Ajan yazı — `docs/reports/OUTBOX/<rol>_done.md`)

```markdown
## ✅ Tamamlanan: <Başlık>

**Tamamlayan:** <Rol>
**Tarih:** DD Ay YYYY
**Durum:** TAMAMLANDI | KISMI | ENGEL VAR

### Yapılanlar
- İşlem 1
- İşlem 2

### Oluşturulan Dosyalar
- path/file.py
- path/component.vue

### Engeller (varsa)
- Engel 1
```

## 🚀 Başlangıç Protokolü

### 1. Team Lead 3 Soru Sorar
```
❓ Proje adı nedir?
❓ Frontend: Vue3, Backend: Django kullanacaksınız? (Y/N)
❓ Database: PostgreSQL mi? (Y/N)
```

### 2. Sistem Kurulur
```bash
$ ./scripts/setup_project.sh     # Klasör yapısı
$ ./scripts/setup_iterm2.sh      # 6 panel + bootstrap
```

### 3. Team Lead Hazırlanır
```
$ docs/project_plan.md yazılır
$ docs/reports/INBOX/*.md görevler yazılır
```

### 4. Ekip Çalışır (Paralel)
```
5 ajan → Bağımsız çalışma → Darboğaz = ❌ YOK
```

## ⚠️ Hata Kontrolü

**Yetkisiz Dosya Yazma:**
```
if dosya_yolu NOT IN agent_write_paths:
    raise ERROR("Dosya yazma yasak. Matrisi kontrol et.")
```

**Eksik Görev:**
```
if INBOX_dosyası EMPTY:
    alert(Agent, "Görev yok, standby moduna geç")
```

**Engel Varsa:**
```
if OUTBOX_durum == "ENGEL VAR":
    alert(TeamLead, "BLOCKER: <engel_açıklaması>")
    TeamLead → Çözüm ara veya başka görev ata
```
